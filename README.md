This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).





## setup

1. github 계정
2. github repo.
3. clone github repo.
4. npm installation globally in macos
5. `yarn create next-app --typescript`
  - zxcv1
  - zxcv2
6. ```
    yarn set version stable # yarn berry
   ``` 
7. `yarn plugin import typescript # yarn typescript 플러그인 설정`
8. `yarn add -D typescript # yarn typescript 플러그인 설정`
9. gitignore 추가
    ```
    .yarn/*
    !.yarn/cache
    !.yarn/patches
    !.yarn/plugins
    !.yarn/releases
    !.yarn/sdks
    !.yarn/versions
    ```
10. gitignore 추가
    ```
    .idea
    ```
